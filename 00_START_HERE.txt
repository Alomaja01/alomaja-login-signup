📞 TROUBLESHOOTING:

Problem: "Page not found"
Solution: Extract ZIP and check all files are present

Problem: "CSS not loading"
Solution: Clear browser cache (Ctrl+Shift+Delete) or check file path

Problem: "PHP not working"
Solution: Ensure PHP is installed (php -v) or use XAMPP/WAMP

Problem: "Login doesn't work"
Solution: Use demo credentials above, not arbitrary passwords

Problem: "JavaScript error"
Solution: Press F12, check Console tab, look for errors

For more help, see: README.md or INSTALLATION.md
