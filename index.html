🚀 Quick Start

  1. Extract the application files to your web server directory
  2. Start your PHP server (Apache, Nginx, or built-in server)
  3. Open index.html in your browser
  4. Click "Sign In" and use demo credentials to test
  5. Click "Create Account" to register a new user
